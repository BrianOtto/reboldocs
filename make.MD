USAGE:
     MAKE :lookahead type def 

DESCRIPTION:
     Constructs or allocates the specified datatype.
     MAKE is a function .

RETURNS: [any-value!]

ARGUMENTS:
    :lookahead [any-value! <...>]
    type [<opt> any-value! <...>]
        The datatype or an example value
    def [<opt> any-value! <...>]
        Attributes or size of the new value (modified)
