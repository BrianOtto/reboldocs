USAGE:
     LIB-MAKE type def 

DESCRIPTION:
     Constructs or allocates the specified datatype.
     LIB-MAKE is a function .

RETURNS: [any-value!]
    Constructed value.

ARGUMENTS:
    type [any-value!]
        The datatype -or- an examplar value of the type to construct
    def [any-value!]
        Definition or size of the new value (binding may be modified)
